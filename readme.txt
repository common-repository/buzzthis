Displays Google Buzz This button for easily sharing an article to google buzz.

== Description ==


* Displays BuzzThis button for easlily sharing a page to Google Buzz. 
* Show the number of buzzes has the pages has got.
* Displayes follow button to easily follow the blogger easily on Google Buzz.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `buzzth_is.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Customize the Google BuzzThis button  from Setting > BuzzThis Button option screen
1. For manual button placement `<?php echo echo buzzth_is(); ?>` in your templates
